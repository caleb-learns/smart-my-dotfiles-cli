# smart-my-dotfiles-cli

My personal dotfiles for bash, vim, git and tmux.

## Install

```bash
git clone https://github.com/caleb-learns/smart-my-dotfiles-cli.git ~/.dotfiles
cd ~/.dotfiles
chmod +x install.sh
./install.sh
```

## Contents

- `.bashrc` — shell aliases, prompt, history settings
- - `.vimrc` — sane vim defaults
  - - `.gitconfig` — git aliases + defaults
    - - `.tmux.conf` — tmux config (mouse mode, 1-indexed panes)
